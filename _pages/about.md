I design and build infrastructure systems for cross-organizational data exchange in agriculture -- platforms that make shared data verifiable, traceable, and trustworthy without a central authority.

My primary project, [DEMETER](/projects/#demeter), is a full-stack blockchain-based platform (Hyperledger Fabric + Go + Node.js + Next.js) that records file provenance across organizations with smart contracts, REST APIs, a natural language interface, and DID-based authentication.

I also built [AI-based computer vision systems](/projects/#marine-cv) for marine aquaculture, automating feeding decisions and size estimation using YOLO, SAM, and OpenCV.

I am looking for infrastructure engineering roles where I can build systems that solve real data problems at scale.
